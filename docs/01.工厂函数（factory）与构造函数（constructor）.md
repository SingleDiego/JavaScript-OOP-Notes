# 01.工厂函数（factory）与构造函数（constructor）

<br>
<hr>
<br>

### 工厂函数（factory）

```
// 工厂函数
function createCircle(radius) {
  return {
    radius,
    draw: function() {
      console.log('draw');
    }
  };
};
```

使用工厂函数新建对象：
```
// 工厂函数的调用
const circle = createCircle(1);
circle.draw();
```

<br>
<hr>
<br>


### 构造函数（constructor）

```
// 构造器函数
function Circle(radius) {
  this.radius = radius;
  this.draw = function() {
    console.log('draw');
  };
};
```

使用构造函数创建新对象：
```
// 构造器函数的调用
const circle = new Circle(1);
circle.draw();
```

注意，构造函数没有 ``return`` 关键字；使用 ``new`` 关键字来创建新对象。
